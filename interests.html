Interests

Software

On my GitHub account I have a couple public projects:

  • A fork of the vim-latex plugin. The idea here—still in progress—is to allow the user to specify plugin parameters on a per-project or per-file basis and to have the plugin's functionality play nicely with the use of some of the parameters for pdflatex
  • This website. I coded this retrograde-looking thing "by hand" in HTML and PHP.

On my personal PC I run Arch Linux, my terminal emulator of choice is st, and my browser is qutebrowser. At some point I'd like to alter qutebrowser so that local files can be read through the user's choice of intermediate program, and so as to use the gopher protocol. I've done the former in a single use case and in a hacky way, running Markdown through pandoc to render it into HTML.
